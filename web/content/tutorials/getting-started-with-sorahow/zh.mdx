<Callout>
在连接自动化流程之前，请先申请 Beta API 访问权限。
</Callout>

## 1. 创建工作区

使用 Sorahow 账号登录并新建工作区，选择「知识库」模板即可自动生成教程与资源栏目。

## 2. 配置多语言

前往 **设置 → 多语言**，启用英文与中文，将英文设为默认语言，并开启「同步结构」确保页面 slug 一致。

## 3. 发布首篇文章

打开教程库，复制「入门模板」，更新 Frontmatter 字段后以 Markdown 或 MDX 撰写正文，最后点击 **发布**。
