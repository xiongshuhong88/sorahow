## Color palette

| Token | Value | Usage |
| ----- | ----- | ----- |
| `primary` | `#38bdf8` | Interactive accents |
| `surface` | `#020617` | Background |
| `muted` | `#475569` | Captions |

## Typography

- **Display:** Space Grotesk 600
- **Body:** Inter 400/500
- **Code:** JetBrains Mono 400

## Components

1. **Navigation bar** – Height 72px, blur backdrop, highlight active locale.
2. **Content cards** – Rounded 20px, border `rgba(148, 163, 184, 0.12)`.
3. **Tag chips** – Uppercase, letter spacing 0.08em, cyan outline.
